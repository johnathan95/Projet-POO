1.Version console
a. Compilation pour la version console du reservation manager
javac ReservationManagerConsole.java

b. Pour executer, on utilise la commande :
java ReservationManagerConsole

2.Version GUI
a. Compilation pour la version avec l'interface graphique du reservation manager
javac ReservationManagerGUI.java

b. Pour executer, on utilise la commande :
java ReservationManagerGUI
